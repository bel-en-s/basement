# Blog Challenge – README

## Overview

This project is part of a frontend challenge to build a blog interface.

## Approach

I approached it from small to large. I started by thinking through each component individually, beginning with the smallest pieces, and applied styles to each one in isolation. As components were ready, I imported them and structured the layout, keeping the hierarchy organized and scalable.

## Design Process

Throughout the process, I constantly compared my implementation with the Figma design. I took screenshots of the deployed version and checked them against the mockups, adjusting small details like spacing, typography, and alignment.

It was an iterative process, fixing small inconsistencies until the result felt as close as possible to the original design.

## Code & SEO

I kept a strong focus on clean code and SEO. I used semantic HTML, maintained a clear component structure, and avoided unnecessary complexity to keep everything readable and maintainable.

## Final Notes

The goal was to balance visual accuracy with solid frontend practices and a clean, scalable structure.
